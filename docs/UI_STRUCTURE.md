## 컴포넌트 목록

### 챗봇 코어 컴포넌트 (Chatbot Core)

* [ ] **`ChatModal`**: 챗봇 전체 컨테이너 (Open/Close 상태 관리)
* [ ] **`MessageList`**: 대화 내용이 쌓이는 스크롤 영역
* [ ] **`MessageBubble`**: 말풍선 (Bot / User 스타일 구분)
* [ ] **`ActionChips`**: 하단 선택지 버튼 그룹 (Quick Reply)
* [ ] **`TypingIndicator`**: 봇 응답 대기 중 '...' 애니메이션

### 추천 결과 컴포넌트 (Result)

* [ ] **`ResultCarousel`**: 추천 가게 카드를 가로로 슬라이드
* [ ] **`StoreCard`**: 가게 정보(이미지, 이름, 평점, 태그) 표시 카드
* [ ] **`TagBadge`**: 추천 키워드(#해장 #얼큰) 강조 뱃지
* [ ] **`MapButton`**: 네이버/카카오 지도 딥링크 연결 버튼

### 공통/유틸 컴포넌트 (Common)

* [ ] **`FloatingActionButton (FAB)`**: 메인 화면 우측 하단 챗봇 트리거
* [ ] **`Header`**: 서비스 로고 및 네비게이션
* [ ] **`LoadingSpinner`**: 데이터 로딩 중 표시
* [ ] **`Toast`**: 위치 권한 오류 등 시스템 메시지 알림

## 인터랙션 가이드

### 1. 챗봇 진입

* **Trigger**: FAB 버튼 클릭 시
* **Mobile**: 화면 하단에서 위로 올라오는 Slide-up 애니메이션
* **Desktop**: 우측 하단에서 페이드인(Fade-in) 되며 팝업

### 2. 메시지 전송

* 사용자가 칩(Chip) 버튼 클릭 시, 해당 텍스트가 유저 말풍선으로 변환되어 리스트에 추가됨 (`Slide-up` 효과).
* 동시에 하단 칩 영역은 사라지고, 봇의 `TypingIndicator` 노출.

### 3. 결과 카드

* 추천 결과가 여러 개일 경우 가로 스크롤(Horizontal Scroll) 지원.
* 카드 스와이프 시 `Snap` 효과로 한 장씩 걸리도록 처리.
