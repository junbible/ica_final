# 컨디션 기반 메뉴 추천 시스템

> "오늘 뭐 먹지?"가 아닌 **"지금 내 컨디션에 맞는 메뉴"** 추천 서비스

## 프로젝트 개요

사용자의 컨디션(피로, 숙취, 스트레스 등)을 파악하여 리뷰 데이터 기반으로 최적의 메뉴와 맛집을 추천하는 서비스입니다.

## 핵심 기능

- **컨디션 기반 추천**: 피로/숙취/스트레스/감기 등 상태에 맞는 메뉴 추천
- **리뷰 분석**: "맵다/든든/깔끔/시원" 등 키워드 추출로 메뉴 특성 파악
- **채팅 인터페이스**: 대화형으로 세부 컨디션 파악

## 기술 스택

| 영역 | 기술 |
|------|------|
| 데이터 수집 | Python, Selenium, Playwright |
| 백엔드 | FastAPI |
| AI/NLP | OpenAI API, KoNLPy |
| 프론트엔드 | React, Next.js |
| 데이터베이스 | PostgreSQL, Elasticsearch |
